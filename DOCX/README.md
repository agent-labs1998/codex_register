```bash
# 开发运行
npm run dev -- --codex-cpa

# 批量注册
npm run batch

# 检查账户
npm run check

# 构建项目
npm run build
```

### 配置文件位置
- 主配置：`config.json`

- TypeScript 配置：`tsconfig.json`
