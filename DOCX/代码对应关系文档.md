## 核心类和函数详解

### 1. OpenAIClient 类 (`src/openai.ts`)

**构造函数参数**：
```typescript
{
  email: string | undefined     // 邮箱地址
  password: string              // 密码
  deviceProfile: DeviceProfile  // 设备指纹
  manualMode: boolean          // 手动输入 OTP 模式
  smsBroker: SMSBroker         // SMS 服务
  bindEmail?: string           // 绑定邮箱
  fetchAddEmailOtp?: () => Promise<string>  // 获取邮箱 OTP
  signupScreenHint?: string    // 注册屏幕提示
}
```

**主要方法**：
```typescript
// 邮箱注册
authRegisterHTTP(): Promise<{authFile?: string}>

// 邮箱登录
authLoginHTTP(): Promise<{authFile?: string}>

// 手机号注册
authPhoneSignupHTTP(
  phone: string,
  fetchOtp: () => Promise<string>
): Promise<{callbackURL: string}>

// ChatGPT Web 登录
authLoginChatGPTWeb(): Promise<void>

// 注册并授权（完整流程）
authRegisterAndAuthorizeHTTP(): Promise<{authFile?: string}>

// 获取 ChatGPT access_token
getChatGPTAccessToken(): Promise<string>

// 保存 ChatGPT access_token
saveChatGPTAccessToken(token: string): Promise<string>
```

---

### 2. SMSBroker 类 (`src/sms/activation-broker.ts`)

**功能**：管理 SMS 号码租赁和验证码接收

**主要方法**：
```typescript
// 获取号码租赁
getActivation(): Promise<ActivationLease>

// 标记失败（触发重试）
markAsFailed(retry: boolean): Promise<void>
```

**ActivationLease 接口**：
```typescript
{
  phoneNumber: string           // 租赁的手机号
  waitForVerificationCode(): Promise<{code: string}>  // 等待 OTP
}
```

---

### 3. 设备配置接口 (`src/device-profile.ts`)

**DeviceProfile 类型**：
```typescript
{
  userAgent: string           // 浏览器 User-Agent
  screen: {
    width: number
    height: number
  }
  language: string           // 语言
  timezone: string           // 时区
  platform: string           // 平台
  // ... 其他浏览器特性

```

## 数据流图

### CPA 模式完整数据流

```
命令行参数 (--codex-cpa)
    ↓
配置加载 (config.ts)
    ↓
准备邮箱 (mail/hotmail.ts)
    ├─ 从卡密池获取邮箱
    ├─ 初始化 IMAP 连接
    └─ 设置 OTP 获取函数
    ↓
获取 SMS 号码 (sms/heroSMS.ts)
    ├─ Hero SMS API 请求
    └─ 获取临时号码
    ↓
手机注册 (openai.ts - authPhoneSignupHTTP)
    ├─ 发送注册请求
    ├─ 等待 OTP
    └─ 完成验证
    ↓
CPA OAuth 流程 (cpa-codex.ts)
    ├─ 请求授权 URL
    ├─ 走 OAuth 登录
    ├─ 提交回调给 CPA
    └─ 从 CPA 拉取 auth 文件
    ↓
获取 ChatGPT Token (openai.ts - getChatGPTAccessToken)
    ↓
试用探测 (probe-trial.ts) [可选]
    ├─ 使用日本代理
    └─ 检查 Stripe 结账
    ↓
保存 Token
    ├─ 写入本地文件
    └─ 追加到 token 池
    ↓
消费 Hotmail 卡密 (consume-hotmail.ts)
    ├─ 从 pool_emails.txt 移除
    └─ 追加到 history 文件
    ↓
启动 GP Plus 订阅 [可选]
    └─ 调用 Python 脚本
```

## 配置文件结构

### config.json 配置映射

```json
{
  // 邮箱提供商配置
  "provider": "hotmail",           → mailbox.ts

  // 代理配置
  "defaultProxyUrl": "socks5://...", → openai.ts

  // 密码配置
  "defaultPassword": "...",         → openai.ts

  // 循环延迟
  "loopDelayMs": 30000,             → index.ts

  // Hero SMS 配置
  "heroSMSApiKey": "...",           → sms/heroSMS.ts
  "heroSMSCountry": 33,             → sms/heroSMS.ts
  "heroSMSMaxPrice": 0.08,         → sms/heroSMS.ts
  "heroSMSPriceTiers": [...],      → sms/activation-broker.ts
  "heroSMSPollAttempts": 15,       → sms/activation-broker.ts
  "heroSMSPollIntervalMs": 3000,   → sms/activation-broker.ts

  // CPA 配置
  "cliproxyApiAutoUploadAuth": true, → cpa-codex.ts
  "cliproxyApiBaseUrl": "...",       → cpa-codex.ts
  "cliproxyApiManagementKey": "...", → cpa-codex.ts

  // 注册资料生成
  "profileLocale": "auto",          → profile-generator.ts / profile-geo.ts
  "profileAgeMin": 25,              → profile-generator.ts
  "profileAgeMax": 34,              → profile-generator.ts
  "profileIpLookupUrl": "https://ipinfo.io/json", → profile-geo.ts
  "profileIpLookupTimeoutMs": 10000,              → profile-geo.ts
  "profileLocaleByCountry": {...}                   → profile-generator.ts
}
```

## 性能优化和并发控制

### 批量注册并发
- 最多 8 次号码重试 (`MAX_PHONE_TRIES`)
- 请求间隔 30 秒 (`loopDelayMs`)
- SMS 轮询间隔 3 秒 (`heroSMSPollIntervalMs`)
- SMS 最多 15 次轮询 (`heroSMSPollAttempts`)

### Token 管理
- 去重检查避免重复写入
- 追加模式而非覆盖
- 原子文件操作
