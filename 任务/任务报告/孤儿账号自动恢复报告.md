### 1.1 新增文件：`src/recover-orphans.ts`

**文件位置：** `A:\Github\codex_register\src\recover-orphans.ts`

**功能：** 孤儿账号自动恢复的核心逻辑模块

**导出接口：**
```typescript
export async function recoverOrphans(options: RecoverOrphansOptions): Promise<RecoverOrphansResult>
```

**核心流程：**
1. 查询 `resolved=0` 的孤儿账号
2. 逐条尝试恢复：
   - 创建新邮箱（`getEmailAddress()`）
   - 登录 OpenAI（`authLoginHTTP()`）
   - 绑定新邮箱（`sendAddEmail()`）
   - CPA 入库（`submitOAuthCallback()`）
3. 成功 → `resolved=1, resolved_note="自动恢复成功"`
4. 失败 → 更新 `resolved_note`，不标记 resolved（允许重试）
5. 每条之间随机延迟 10-30 秒

**行数：** 约 120 行


### 3.3 错误处理

- **邮箱池耗尽** → 提前终止，跳过剩余账号
- **账号不存在/密码错误** → 标记为 `resolved=1`，备注"无法恢复"
- **其他错误** → 更新备注，不标记 resolved，允许下次重试
